# FORM_REQUIREMENTS.md

## Form Goal

Collect qualified consultation leads without creating too much friction.

## Required Fields

- Name
- Phone Number
- Email
- Project Type
- City / Area
- Brief Project Description

## Optional Fields

- Timeline
- Is the project already submitted to the city?
- Preferred contact method

## Form CTA

Primary:
Start My Permit Process

Alternatives:
Request My Free Consultation
Get a Permit Review

## Form UX Rules

- Form must be visible above the fold on desktop.
- On mobile, CTA buttons should scroll to the form.
- Keep labels clear.
- Use large tap targets.
- Show privacy/consent text.
- Use success and error states.
- Do not make the form feel intimidating.
- Do not ask for budget in the first version unless required.

## Suggested Microcopy

“Not sure what permits you need? Send us your project details and we’ll help you understand the next step.”

Consent text:
“By submitting this form, you consent to be contacted about your project request.”
