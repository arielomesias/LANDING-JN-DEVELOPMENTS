# DESIGN_SYSTEM.md

## Visual Direction

Premium permit expediting / construction consulting / architecture office.

The page should feel:
- modern
- clean
- high-trust
- local
- professional
- slightly premium

## Layout Style

Use strong sections with clear hierarchy.
Use generous spacing.
Use cards for services and project types.
Use a sticky CTA on mobile if appropriate.

## Suggested Visual Motifs

- architectural grid lines
- blueprint-inspired details
- subtle permit stamp elements
- city map / LA + OC service area cues
- document review / plan check visual language

## Colors

Use a professional palette:
- deep navy or charcoal
- warm off-white
- construction gold or muted orange as accent
- light gray backgrounds
- green only for success/proof elements, if needed

## Typography

Use a strong modern sans-serif.
Headings should be bold and readable.
Body text must be highly readable on mobile.

## Animations

Animations should be subtle:
- fade up on section entry
- card hover lift
- smooth scroll to form
- animated counters only if values are real and sourced
- lightweight blueprint line motion if performance is not affected

Avoid:
- distracting parallax
- excessive spinning
- heavy 3D
- animations that slow down mobile
