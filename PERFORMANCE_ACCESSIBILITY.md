# PERFORMANCE_ACCESSIBILITY.md

## Performance Requirements

- Mobile-first.
- Fast loading.
- Avoid heavy animation libraries unless necessary.
- Optimize images.
- Lazy-load below-the-fold images.
- Keep JavaScript minimal.
- Do not use large background videos in the first version.

## Accessibility Requirements

- Semantic HTML.
- Proper heading hierarchy.
- Sufficient contrast.
- Visible focus states.
- Labels for form fields.
- Buttons must be keyboard accessible.
- Images must have useful alt text.
- Avoid text embedded inside images.

## Core Web Vitals Goal

The landing page should feel instant on mobile.
Animations should not block interaction.
