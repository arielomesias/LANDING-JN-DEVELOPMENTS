# QA Review Skill

Use this skill before considering the page complete.

## Review Checklist

1. Check mobile layout.
2. Check form usability.
3. Check CTA consistency.
4. Check phone number.
5. Check heading hierarchy.
6. Check loading performance.
7. Check accessibility.
8. Check copy clarity.
9. Check that no unsupported claims were invented.
10. Check that every section supports conversion.

## Output

When reviewing, return:
- Critical issues
- Conversion issues
- Design issues
- Technical issues
- Suggested fixes
