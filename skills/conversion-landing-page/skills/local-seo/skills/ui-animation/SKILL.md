# UI Animation Skill

Use this skill when adding animations to a conversion-focused landing page.

## Animation Principles

- Animations must support clarity and trust.
- Keep motion subtle and premium.
- Prioritize performance.
- Respect reduced motion preferences.
- Avoid distracting effects near the form.

## Recommended Effects

- Fade-up section reveals.
- Smooth anchor scrolling.
- Gentle card hover states.
- Subtle blueprint/grid background movement.
- Lightweight CTA hover animation.

## Avoid

- Heavy parallax.
- Large video backgrounds.
- Excessive motion.
- Animations that delay content visibility.
