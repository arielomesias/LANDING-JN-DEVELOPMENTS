# Local SEO Skill

Use this skill when optimizing landing pages for local service businesses.

## Workflow

1. Identify primary service keyword.
2. Identify target locations.
3. Add local relevance naturally.
4. Add service-area section.
5. Add FAQ section.
6. Add LocalBusiness, Service, and FAQ schema where appropriate.
7. Write meta title and meta description.
8. Check heading structure.

## Rules

- Do not keyword stuff.
- Do not create fake addresses.
- Do not invent service areas.
- Keep SEO secondary to conversion unless instructed otherwise.
